# Portfolio Website

This is my personal portfolio website created using HTML and CSS. It showcases my projects, basic information about me, and contact details.

## 🔗 Live Website
https://avinashmenon151-art.github.io/portfolio/

---

## 📁 Website Structure

The project is organized as follows:
```
portfolio/
│── index.html        # Main homepage
│── projects.html     # Projects details page
│── style.css         # Styling for entire website
│── images/           # Contains all project images

```


---

## 📄 Pages Description

### 1. Home Page (index.html)
- Introduction (Hero Section)
- About Me section
- Project preview section
- Contact section
- Navigation menu and footer

### 2. Projects Page (projects.html)
- Displays detailed project cards
- Includes images, descriptions, and links to GitHub
- Navigation back to homepage

---

## 🎨 Features

- Responsive design using CSS and media queries
- Navigation menu with multiple links
- Project showcase with images and descriptions
- Smooth animations and hover effects
- Typing animation on homepage
- Clean and modern UI design
